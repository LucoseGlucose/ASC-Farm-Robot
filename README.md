# ASC-Farm-Robot

![Machine with Cow Inside](Media/Cow Inside.jpg)

Final project for SAMET, SUNY Alfred State College of Technology course ELET 7404 Embedded and Real Time Systems. The goal was to create a scaled down prototype of a commercial DeLeval milking machine like the one at the college farm. The robot features a full 3D printed frame and arm assembly, RFID tag tracking system, powerful arm with inverse kinematics, and pump placement sensing.
